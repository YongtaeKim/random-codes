Compile instruction

for Server Program:
>javac projectSWServer.java Server_enemyLaser.java Server_enemyPlayer.java Server_Image.java Server_Laser.java Server_Player.java Server_Window.java

for Client Program:
>javac Client_enemyLaser.java Client_enemyPlayer.java Client_Image.java Client_Laser.java Client_Player.java Client_Window.java projectSWClient.java

-----------------------------------------------------------

Run instruction

for Server Program:
>java projectSWClient [Server IP] [Server port]

for Client Program:
>java projectSWServer [port]
- Then wait until the Client side connects.

----------------------------------------------------------

How to play:
Move with MOUSE, Press SPACEBAR to fire a laser.

Programmed to continuous run until player choose to exit.
